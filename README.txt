【运行方式】
运行方法就是点击AutoDrawer.exe直接运行就好！
因为很多人反馈没法画，经过追踪发现是因为缺少管理员权限。
我欲画画，系统竟然不许？！那可不行，所以代码里直接加入了强制获取管理员权限，解决这个问题~
【设置调整】
线条细节越往左细节越多，解析和绘画的时间也越长
简单出图效果的话，不妨直接拉到最右
顶部避开调整为10%最佳，如果遗物多了的话要规避可以再调高些
当然，现在你可以直接手框范围了，不用自己手动调规避
追求绘制速度还可以把拖拽步长调高，但是注意调高后可能会导致画面崩坏
出现飞线问题的话可以试试修改起落笔延迟，然后尽可能调高游戏帧率（不是显示帧率，程序只跟游戏逻辑帧有关）
【关于自动对齐】
自动对齐只要你大概把地图挪到原画附近他就会开始自己找了。
能近一点近一点，太远了找不到就直接重画咯~
【迷雾战场模式】
经过大家意见加入的新功能，可以自动把所有图标都涂了！
宝箱我留下来没涂，
一方面线条颜色跟地图上偶有的污渍颜色接近容易造成识别错误
另一方面必然出现且位置固定没必要遮
仍然可以通过F8暂停与继续！
速度较慢请见谅...如果要玩迷雾模式的话还请提前跟朋友们说哦？
当然，其实也可以偷偷涂...
毕竟自动从上往下涂抹的，等他们打完一二回地图看的时候已经涂的差不多哩！
